# DDing(띵) - Visual Safety Awareness Tool 👁️
- 화면과 진동을 통해 알려주는 **실시간 위험 감지 앱**

## 🎯 프로젝트 개요
- **타겟**: 시각 장애인, 청각 장애인, 스마트폰 보행자
- **기술**: Google AI Studio 
- **기간**: 2026.02.26~03.01

## ✨ 핵심 기능
✅ 실시간 위험 감지 (레이더 UI + START 버튼)
✅ 햅틱 피드백 (Device Vibration API)
✅ 감지 기록 (HISTORY 탭)
✅ 사용 가이드 (GUIDE 탭)
✅ 모바일 반응형 디자인

## 🎓 배운 점
1. CSS Grid + Flexbox 복합 레이아웃
2. 반응형 디자인 (모바일 우선)
3. 접근성 고려한 UI/UX
4. SPA 네비게이션 구현
